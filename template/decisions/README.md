# 决策记录

> 记录个人决策及复盘，用于定期回顾和反思。

## 格式

每条决策一个文件，文件名：`YYYY-MM-DD-简短标识.md`

```markdown
---
title: 决策标题
date: YYYY-MM-DD
tags: [投资/职业/生活]
status: open           # open | reviewed | closed
review_date: YYYY-MM-DD
context: analysis/xxx   # 关联的研究（如有）
---

## 决策
做了什么决定。

## 当时理由
为什么这样做。

## 预期
期待什么结果。

## 复盘
（待复盘）
```

## 状态说明

- `open`：待复盘（review_date 到期后由复盘流程处理）
- `reviewed`：已复盘
- `closed`：决策已结束，不再跟进

## 来源

- 研究产出决策型结论时自动创建
- 用户手动说"记录一个决策"
