# 跨课题关联

这里存放跨课题关联分析卡片。每张卡片记录不同研究课题之间的关联发现，产出单篇研究得不到的新洞察。

## 与 topics 的区别

- **topics/** = 专题内聚合（同标签文章汇总）
- **connections/** = 专题间关联（跨课题交叉分析，A + B → C）

## 关联类型

| 类型 | 含义 | 示例 |
|------|------|------|
| causal | 因果链（A 导致 B） | 油价上涨 → PDD 物流成本上升 |
| resonance | 共振（不同领域同步变化，共享深层驱动） | FSD 推迟 + Robotaxi 19% = AI 叙事退潮 |
| contradiction | 矛盾（不同课题结论冲突） | 宏观"滞胀" vs 高通"汽车对冲" |
| hedge | 对冲（A 利好 = B 利空） | 高通数据中心成功 → Cerebras 溢价削弱 |

## 卡片格式

文件名：`YYYY-MM-DD-简短标题.md`

```markdown

created: YYYY-MM-DD
type: causal | resonance | contradiction | hedge
topics: [课题1, 课题2]
sources: [相关文件路径]
---

- 课题A：...
```

## 触发方式

不单独触发，嵌入已有流程：
- 研究过程中自然产出（深度）
- 文章入库时轻量扫描（增量）
- Sweep 定期扫描（兜底）
