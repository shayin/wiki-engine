# 聚合主题（topics/）

当同一标签下的知识卡片积累到 **≥3 篇**时，自动生成主题页面，跨文章聚合核心发现。

## 生成规则

- 触发条件：同一标签 ≥3 篇卡片
- 由 `/wiki-digest` 处理文章时或 `/wiki-sweep` 扫描时自动检查并生成
- 已有主题在新文章入库时自动更新

## 主题格式

```markdown
---
title: 主题名称
tags: [标签]
updated: YYYY-MM-DD
related: [[文章1]], [[文章2]]
---

## 概述
基于 N 篇文章的聚合总结。

## 核心发现
- ...

## 关联文章
- [[文章1]] - 一句话摘要
- [[文章2]] - 一句话摘要
```

## 与 sources/ 的区别

- `sources/`：单篇文章的提炼，一对一
- `topics/`：多篇同主题文章的聚合，多对一
