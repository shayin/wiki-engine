# Plan: 跨课题关联（Connections）

## 状态：待确认

## 核心理念

知识库的核心差异点不是单篇文章内容（搜索引擎都有），而是：
1. **独特信息源** — 个人经验、一手调研
2. **内容关联创造** — 不同信息交叉分析产生新洞察

架构三层递进：
- **sources/** = 单篇文章（原子级）
- **topics/** = 专题内关联（同标签聚合）
- **connections/** = 专题间关联（跨课题新洞察，A+B→C）

## 新增目录

```
wiki/connections/
├── README.md                   # 目录说明（用途、格式、触发方式）
├── index.md                    # 关联索引（按类型分组）
└── YYYY-MM-DD-简短标题.md      # 关联分析卡片
```

## 关联分析卡片格式

```markdown
---
title: 油价上涨削弱 PDD 毛利率安全垫
created: 2026-05-18
type: causal          # causal | resonance | contradiction | hedge
topics: [美国宏观经济, 拼多多]
sources: [wiki/analysis/美国宏观经济/report.md, wiki/analysis/拼多多/report.md]
---

## 关联发现
WTI 涨至 $102+ → TEMU 跨境物流成本上升 → 毛利率可能进一步压缩 → 安全垫比单独评估更薄

## 具体分析
- 美国宏观：霍尔木兹海峡受阻，运费上涨
- 拼多多：TEMU 出海成本已导致毛利率从 62.6% 降至 55%
- 交叉推导：油价每涨 $10，跨境物流成本约 +2-3%

## 对现有认知的影响
- 拼多多：估值底部 argument 需打折扣
- 美国宏观：油价传导链增加了中概电商尾部风险
```

## 关联类型

| 类型 | 含义 | 示例 |
|------|------|------|
| causal | 因果链（A导致B） | 油价上涨 → PDD物流成本上升 |
| resonance | 共振（不同领域同步变化） | FSD推迟 + Robotaxi 19% = AI叙事退潮 |
| contradiction | 矛盾（结论冲突） | 宏观"滞胀" vs 高通"汽车对冲" |
| hedge | 对冲（A利好=B利空） | 高通数据中心成功 → Cerebras溢价削弱 |

## 触发机制

### 触发点 1：研究过程（深度产出）
- 修改 wiki-research SKILL.md
- 分析完成后检查核心发现是否与其他课题有关联
- report.md 增加 `## 跨课题关联` 节
- 发现关联时创建 connection 卡片 + 更新 index.md

### 触发点 2：文章入库（轻量扫描）
- 修改 wiki-digest SKILL.md
- 现有"关联发现"步骤扩展：增加跨专题语义关联检查
- 轻量级：只记录关联线索，不做深度推导
- 无关联则跳过

### 触发点 3：Sweep 扫描（兜底补漏）
- 修改 wiki-sweep SKILL.md
- 新增维度 K：关联发现
- 扫描所有 topics/ 和 analysis/*/report.md
- 重点检测：共享驱动因素、因果链、对冲关系
