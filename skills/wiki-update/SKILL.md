# Update — 课题数据更新

## 触发条件

- "更新 XX"、"刷新 XX"、"XX 更新一下" → **全量更新**
- "更新 XX 的 YY"、"刷新 XX 的 YY" → **定向更新**（只刷新指定数据）
- 示例："更新美国宏观经济"、"刷新高通的 RSI 和股价"、"更新拼多多的财报数据"

**缺少课题名时**：列出已有课题让用户选择。

## 前置检查

1. **获取准确时间**（必须）：执行 `date "+%Y-%m-%d %H:%M %a"` 获取当前日期、时间和星期，作为数据时效性判断的锚点
2. 检查 `WIKI_ROOT` 环境变量是否设置
3. 定位课题目录：`$WIKI_ROOT/wiki/analysis/{课题slug}/`
4. 课题目录不存在时：列出 `wiki/analysis/` 下所有课题让用户选择

## 执行步骤

### 1. 读取现有报告

读取 `report.md`，提取所有量化数据点：

- 带有数值的陈述（股价、指标、财务数据、增长率等）
- 已有的 `[截至 YYYY-MM-DD]` 标注
- `## 数据时效性说明` 表（如果存在）

### 2. 判断需要更新的数据

按 CLAUDE.md 时效窗口规则判断：

| 数据类型 | 时效窗口 | 是否需要更新 |
|---------|---------|-------------|
| 市场价格、行情 | 超过 1 周 | 是 |
| 经济指标 | 超过 1 个月 | 是 |
| 公司财务数据 | 超过 1 个季度 | 是 |
| 行业趋势 | 超过 1 个月 | 是 |
| 方法论、原理 | 长期 | 否 |

**全量模式**：列出所有超过时效窗口的数据点
**定向模式**：只列出用户指定的数据类型相关数据点

向用户展示需要更新的数据清单，确认后继续。

### 3. 搜索最新数据

对每个需要更新的数据点：

1. 构造搜索查询（包含指标名、公司名、时间段）
2. 用 WebSearch 搜索最新数据
3. 验证：交叉检查 2-3 个来源，标注来源可信度
4. 记录新旧对比

搜索策略：
- 优先搜索官方数据源（BLS、SEC、公司 IR 页面）
- 多个独立查询可并行
- 搜索失败的标注"未获取到最新数据"

**补充搜索层**（每次更新必须执行，不可跳过）：
- **中文源扫描**：至少用中文搜一次"标的+最新/动态/新闻"，覆盖华尔街见闻、富途资讯、36氪、雪球等中文源
- **SEC/监管文件**：涉及美国上市公司时，检查 SEC EDGAR 最新 8-K/13F/SC 13D 等文件
- **产业链验证**：搜索标的的主要供应商/客户/竞争对手的最新动态，从产业链侧面验证叙事
- **叙事变量扫描**：除刷新已知变量外，快速搜索是否有新的叙事变量出现（新项目、新合作、新诉讼等）
- **跨领域交叉搜索**：搜索标的+半导体/AI/芯片等跨界关键词，不只搜标的本身
- **行业分析师渠道**：用知名分析师名+标的组合搜索（如"郭明錤 Tesla"、"Dan Ives TSLA"）

### 4. 更新报告

在 `report.md` 中更新数据：

**数据更新规则**：
- 保留原有结论结构，不重写报告
- 更新的数据点标注 `[截至 YYYY-MM-DD, 原值: XX]`
- 新数据推翻旧结论时，在结论后追加 `> ⚠️ 更新于 YYYY-MM-DD：原结论基于 [旧数据]，最新数据显示 [新情况]，结论修正为 [新结论]`
- 不推翻旧结论时，只更新数据值和截至日期

**更新 `## 数据时效性说明` 表**（如果不存在则在 `## 未解决问题` 后新增）：

```markdown
## 数据时效性说明
| 核心数据 | 截至 | 状态 | 更新记录 |
|---------|------|------|---------|
| 2Y-10Y 利差 | 2026-05 | ✅ 已更新 | 原 2025-07 (+50bps) → 2026-05 (+XXbps) |
```

### 5. 更新材料（可选）

如果搜索到重要的新数据，且现有 materials/ 中缺少覆盖：

- 创建新材料文件 `materials/NNN-简短标题.md`
- frontmatter 包含 `url`、`title`、`fetched`、`data_as_of`
- 在 report.md 中更新对应的来源引用

### 6. 更新日志

在 `wiki/log.md` 追加更新记录：

```
- YYYY-MM-DD HH:mm: 更新了 {课题名} 的 N 个数据点（{数据点列表}）
```

## 输出格式

向用户展示更新摘要：

```
{课题名} 更新完成：

📊 数据更新：
- WTI 原油：$96 → $102-106（+6-10%，截至 2026-05）
- 4月 CPI：预期 3.8% → 实际 3.8%（已确认，BLS 5/12 发布）
- S&P 500 YTD：+8.22%（截至 5/16，未变化）

⚠️ 结论修正：
- [高通] RSI 从 85.82 降至 ~70，"深度超买"结论不再成立，加仓的技术面阻碍已消除

✅ 确认不变：
- 美伊冲突仍是核心变量，霍尔木兹海峡局势无重大变化
```

## 错误处理

- **课题不存在**：列出可用课题供选择
- **课题无 report.md**：提示该课题未完成研究，建议用 `/wiki-research` 继续
- **搜索无结果**：标注"未获取到最新数据"，保留原值并标注截至日期
- **新数据与旧数据矛盾**：标注冲突，保留两者，提醒用户判断
